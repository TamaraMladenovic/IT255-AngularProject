# Angular Flight Booking Application

A web application developed using **Angular** as part of the **IT255** course project.  
The application provides functionality for managing flight reservations through standard CRUD operations.

## Application Features
- Create new flight reservations
- View available flights and existing reservations
- Update reservation details
- Delete reservations
- Structured navigation between application views

## Application Overview
The system allows users to manage flight bookings by entering passenger and flight details.  
All core functionalities are implemented using CRUD principles, focusing on data manipulation and user interaction through the web interface.
